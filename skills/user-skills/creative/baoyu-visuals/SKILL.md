# Baoyu Creative Visuals

Creative visual generation tools by 宝玉 (JimLiu). Three modes for different visual storytelling needs:

| Mode | Trigger | Reference |
|------|---------|-----------|
| **Article Illustrator** | Blog post, article, tutorial needs inline images | `references/article-illustrator/SKILL.md` |
| **Knowledge Comic** | Educational content, biography, tutorial as comic | `references/comic/SKILL.md` |
| **Infographic** | Data visualization, summary, comparison | `references/infographic/SKILL.md` |

All modes share the same **Type × Style × Palette** consistency model. Load the relevant reference for the mode you need.
