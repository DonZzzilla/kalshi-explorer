# BOA Discord Command Reference
# For use with Hermes Agent on Zzzilla Island Discord

## Wiki & Build Commands
| Command | What it does |
|---------|-------------|
| `!got` | Trigger GoT wiki build/update cycle |
| `!csez` | Trigger CSEZ wiki build/update cycle |
| `!boa` | Trigger BOA wiki build/update cycle |
| `!silentnorth` | Trigger Silent North wiki build/update cycle |
| `!all` | Run full wiki maintenance sweep on all wikis |
| `!status` | Show status of all wikis (pages, users, last edit) |

## Tactical & Game Info Commands
| Command | What it does |
|---------|-------------|
| `!tac` | Random tactical tip for Ghosts of Tabor |
| `!entry` | Quick ref: 3-man entry techniques |
| `!aor` | Quick ref: AOR coverage for 2/3-man squads |
| `!breach` | Quick ref: breach decision matrix |
| `!callouts` | Quick ref: standard callouts & comms |
| `!fm [number]` | Show specific field manual (e.g. `!fm 02` for Room Clear) |
| `!sop` | Quick ref: pre-raid checklist & debrief |

## BOA Operations Commands
| Command | What it does |
|---------|-------------|
| `!roster` | Show BOA member list & ranks |
| `!schedule` | Upcoming BOA sessions & events |
| `!tactics` | Link to Tactical BOA Operations wiki section |
| `!newplayer` | Link to Beginner FAQ + Getting Started guides |
| `!rules` | Link to BOA Discord Rules & Policies |

## Utility Commands
| Command | What it does |
|---------|-------------|
| `!patches [days]` | Show recent GoT patch notes (default 7 days) |
| `!patchnotes [version]` | Show specific patch notes (e.g. `!patchnotes 0.9.0`) |
| `!map [name]` | Quick map info (e.g. `!map silo`, `!map mall`) |
| `!weapon [name]` | Quick weapon stats (e.g. `!weapon awm`, `!weapon saiga`) |
| `!ammo [type]` | Quick ammo stats (e.g. `!ammo 7.62`, `!ammo 12ga`) |

## Admin Commands
| Command | What it does |
|---------|-------------|
| `!purgewiki [wiki]` | Run full maintenance sweep (GoT/CSEZ/BOA/SN) |
`!rebuildnav [wiki]` | Rebuild navigation templates |
| `!checklinks [wiki]` | Scan for broken links |
| `!checkfiles [wiki]` | Scan for broken file/image links |
| `!categorize [wiki]` | Run batch categorization |
