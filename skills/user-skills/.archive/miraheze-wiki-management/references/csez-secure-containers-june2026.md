# CSEZ Secure Containers (Save Boxes)

Created 2026-06-03 from RadFox University transcript (https://www.youtube.com/watch?v=XV9OBww4tI8) and DLC review (https://www.youtube.com/watch?v=h3UpxrP8SvU).

## Mechanics

- **Name:** "Save Box" or "Safe Box" (NOT "Secure Container" — that's GOT terminology)
- **Access:** Stored on player's lower back. Reach behind to grab and open.
- **Key feature:** Items inside have **zero weight** — ideal for heavy valuables (gold ingots, GPUs, lamps, golden cups)
- **Quest gate:** All base game Save Boxes unlocked via Johnny's quest progression
