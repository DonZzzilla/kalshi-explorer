# Fauna Report Format (June 2026)

The "Daily Animal Fact Email" cron job sends a daily fauna report. Don specified the format and content rules:

## Structure

Each report follows this template:

```
🦁 **Daily Fauna Report: [Animal Name]**
[Common name] — [Brief one-liner hook]

[3-4 interesting facts about behavior, habitat, unique adaptations, hunting style]

**Did You Know?** [One surprising closing fact]

**🎮 Ghosts of Tabor Tactical Connection:**
[Creatively connect the animal's behavior/nature to Ghosts of Tabor gameplay — 
weapons, tactics, map play, squad behavior, BOA doctrine. Make it clever and fun.]
```

## Rules

1. **Three animal categories rotate:** Spiders, Snakes, Random Creepy Crawlers
2. **Tactical connection is KEY** — the whole point is linking animal behavior to Ghosts of Tabor gameplay
3. **NEVER reference "Game of Thrones"** — the tactical connection is to Ghosts of Tabor the GAME, not the TV show
4. **Use correct game terminology:**
   - GoT = Ghosts of Tabor (the VR tactical shooter)
   - BOA = Battlefield Observation & Awareness
   - CWS = Combat Waffle Studios (the developers)
5. **Family-friendly** content — no gore, no inappropriate references
6. **Under 1200 characters** for Telegram delivery
7. **Use emoji sparingly** but effectively (🦁 🐍 🕷️ 🎮)

## Thematic Connection Style

The tactical section should:
- Map animal behavior to specific GoT gameplay mechanics (weapons, movement, positioning)
- Reference BOA doctrine where relevant (AOR, stack, breach, extract)
- Be creative and surprising — the reader should think "that's actually a great analogy"
- Use GoT-specific terms: K/D, ADS, peek, flank, rat, chad, kit, extract, etc.
- Match the tone of the community (casual, self-aware, slightly irreverent)

## Example References

- Peregrine Falcon → AWM player holding power angle (precision, patience, one shot)
- Mantis Shrimp → shotgun playstyle (close range, full send, no aim)
- Honey Badger → reckless playstyle (no fear, fights everything)
- Octopus → flexible positioning, camouflage, unexpected angles
- Honey Bee → sacrifice play (sting kills you but damages the enemy)
