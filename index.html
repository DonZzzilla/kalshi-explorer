💡 What is Kalshi?

Kalshi is a prediction market where you trade on the outcome of real-world events. Each market has a YES and NO price in cents (1-99). If you buy YES at 65¢ and the event happens, you get $1.00 per contract. If not, you lose your 65¢.

Example: "Dow Above 45000" — YES price is 12¢
This means the market thinks there's a 12% chance the Dow closes above 45000.
Buy 100 YES contracts at 12¢ each = $12 invested. If Dow > 45000, you get $100. Profit = $88.

💡 Earnings: What Could I Make?

Imagine you find a market where the YES price is 70¢. That means you pay 70¢ per contract. If you're right, your contract pays $1.00. If you're wrong, you get nothing.

Think of it like this:
• Price = what you pay for a ticket
• $1.00 = what the ticket pays if you win
• Profit = $1.00 minus what you paid
So at 70¢, every winning ticket makes 30¢ profit. But you lose your 70¢ on every losing ticket.

  • • Browsers block cross-site requests for security (called CORS)
  • • Kalshi's API doesn't allow browser access yet
  • • So we use fake-but-realistic data to show how it works
  • • A server-side bot CAN hit the real API with no issues

📝 API ENDPOINTS YOU CAN USE

These are the URLs a server-side bot would call. No login needed!

GET /v2/markets — List all markets
GET /v2/markets/{ticker} — Single market detail
GET /v2/events — List all events
GET /v2/events/{id}/markets — Markets in an event

Base URL: https://external-api.kalshi.com
Auth: None • Rate limit: ~100 requests per minute
